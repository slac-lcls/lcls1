## Quick start
### Create conda release
See for detail Psana Developer Documentation [3] 
```
cd <my-conda-release>
source conda_setup
```

### Clone package
**on pslogin:**
```
git clone https://github.com/lcls-psana/Detector.git
# or 
condarel --addpkg --name Detector --tag HEAD
```
### Build 
```
scons
```
